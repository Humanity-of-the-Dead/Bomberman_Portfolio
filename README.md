# カブーム・アリーナ
2D対戦アクション。
爆弾を設置して相手を倒す対戦ゲームです。

## 開発環境/制作体制
* **制作人数**:2人
* **エンジン**:Unity(C#)
* **制作時間**
  * **期間**:2025/10/2~2025/11/6
  * **作業時間**:約23時間
* **役職**:リードプログラマ

## 担当箇所
* Player移動
* シーン遷移時に暗いままのバグ修正
* 軽量化
* 爆弾設置アクション
* BGM/SE選定

## 操作方法
Joy-Con(2本)の横持ちに対応しています。
* **移動**:スティック
* **爆弾設置**:Y/左
